# Branding Drafts

This directory stores early branding assets for the VPN website.

Current files:

- `free-access-logo-v1.svg`: static icon mark
- `free-access-logo-v1-animated.svg`: SVG animation with a moving signal tracer
- `freedom-safety-style-animated-v1.svg`: higher-drama neon hero treatment inspired by the later static concept

Design intent:

- Open globe: global reach
- Flowing route: open access across the network
- Network nodes: modern protocol and routing infrastructure
- Upward exit path: freedom of movement rather than a closed system

Current palette:

- Navy: `#0F172A`
- Deep blue: `#1E3A8A`
- Cyan: `#22D3EE`
- Sky blue: `#38BDF8`

Suggested next steps:

- Add a horizontal lockup once the final site name is chosen
- Create a monochrome version for favicons and small avatars
- Create a dark-background variant for headers and app icons
